### 注册

>POST /Api/Customer/join.aspx

|  字段          |        说明        |
| -------------  | ------------------ |
|   Tel          | 用户已存在 |
|   Password     | 给true或者false    |
|   IsNeedPromoter | 是否需要推荐人第一次请求IsNeedPromoter=1|
|   PromoterTel    | 推荐人电话                       |

返回信息

|   state        |        内容        |
| -------------  | ------------------ |
|   -3   | 用户已存在                 |
|   -1   | 推荐人不存在               |
|   0    | 推荐人没有开能推荐功能     |
|   1    | 成功                       |

### 设置用户是否为推荐人

>POST /Api/Customer/SetIsPromoter.aspx

|  字段          |        说明        |
| -------------  | ------------------ |
|   userId       | 用户已存在         |
|   isPromoter   | 给true或者false    |
|   0    | 推荐人没有开能推荐功能     |
|   1    | 成功                       |

返回

|返回值|内容|
|---|---|
|1|更新更成功|
|0|userid不存在|
